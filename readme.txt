== Description ==

This plugin creates a widget area on the bottom right corner of your site, with a 'toaster' like effect.

Features

* Edit title
* Adjust size (width/height)

Coming soon features

* Choose animation
* Choose colors
* Specify individual URLs
* Open on user leaving the page


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload plugin to `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done!

== Screenshots ==

1. Example of the 'toaster' looks opened
2. Example of the 'toaster' looks closed
3. In the settings page you can configure the title and the width/height
4. To place any widget you want go to 'appearance > widgets'. There will be a area called 'Toaster Widget' where you can just drag and drop your widgets.
